# Permafrost-Methane-MultiTracer-and-Monte-Carlo-Simulation

**List of files:**
PermafrostCH4_MultiTracerMC.m run multi-tracer inversions and simulates the permafrost CH4 uncertainties through Monte Carlo.
PermafrostCH4_SensitivityTest.m run above simulation the same way but plot the uncertainty from each individual parameter.
